# scrape-app
All the News That's Fit to Scrape

## How to use:
- This app scrapes a news site with just a click.
- Now the collection of articles may be 'saved' and even notes can be added to list of articles.

## Technologies:

express, 
express-handlebars, 
mongoose, 
cheerio, 
axios
